## Git
### インストール
- sudo apt install git
### 初期設定
- git config --global user.name "name"
- git config --global user.email "address"
### CONFIG
- git config -l
### PULL
- git pull 
### ADD
- git add ファイル名
### COMMIT
- git commit -m "comment"
- git commit -F ファイル名
### PUSH
- git push
### CHECKOUT
- git checkout BRANCH
- git checkout TAG
### STATUS
- git status
### DIFF
- git diff
### TAG
- git tag
### LOG
- git log
### BRANCH
- git branch
### SHOW
- git show コミットID
### CLEAN
- git clean -f
### APPLY
- git apply パッチ
